Software Projects

  • May 21, 2025
  • software-projects  

Below is a list of software projects I’ve created, each with its name and a brief description:

  • kovan – kovan is an eBPF-based network security auditing tool (under development).
  • inject – inject is a command-line network tool for crafting, injecting, and sniffing a wide range of protocols.
  • svsocks – svsocks is a lightweight, fast SOCKS5 server.
  • mhttp – mhttp is an epoll-based minimal HTTPS web server.
  • heyelan – heyelan is a denial-of-service testing tool supporting many attack types.
  • cloglib – cloglib is a small, easy-to-use, fast C/C++ logging library.
  • fmemt – fmemt is a dynamic memory tracker for monitoring heap usage and detecting leaks.
  • games – a collection of small games I made for fun.
